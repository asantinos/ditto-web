## 1. Install

Download `Ditto-Setup-1.1.0.exe` from the [Download page](/download) and run it. Ditto installs to your user folder. No admin rights needed.

<Callout type="note">
Windows SmartScreen will show a warning the first time you run the installer. Click **More info → Run anyway**. The installer is not yet code-signed, which is normal for early open source builds.
</Callout>

## 2. Pick a model

The first time Ditto launches, a Welcome window appears. Pick a Whisper model and Ditto downloads it from HuggingFace to your `%APPDATA%\ditto\models\` folder.

| Model    | Size   | Speed   | Use case |
| -------- | ------ | ------- | -------- |
| Tiny     | 75 MB  | Fastest | Quick notes, lowest latency |
| Base     | 142 MB | Fast    | Casual use, balanced |
| Small    | 466 MB | Medium  | Daily use, better accuracy |
| Medium   | 1.5 GB | Slower  | High accuracy, more CPU/GPU work |
| Large-v3 | 2.9 GB | Slowest | Maximum quality |

<Callout type="tip">
Start with Base. You can switch to a larger one later from **Settings → Models** without reinstalling.
</Callout>

When the download finishes, the Welcome window closes itself and the floating pill appears on screen.

## 3. Your first transcription

1. Click into any text field — a chat, a browser tab, your code editor, anything.
2. Hold <kbd>Ctrl</kbd>+<kbd>Windows</kbd>.
3. Talk, then release the keys to stop and transcribe.

For hands-free recording, quickly press <kbd>Ctrl</kbd>+<kbd>Windows</kbd> twice. Press it once more when you're done.

A second or two later, your transcription is pasted where your cursor was.

<Callout type="note">
The pill shows what Ditto is doing: idle is a thin bar that expands to "Ditto" on hover, recording reacts to your voice, and transcribing shows a compact dot loader. If you don't see the pill, it may be hidden — open it from the tray icon, or keep using the shortcut; it works either way.
</Callout>
